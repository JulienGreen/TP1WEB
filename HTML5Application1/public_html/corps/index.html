Corps

Se débarrasser du corps

L'un des plus gros problèmes rencontrés est de se débarasser du corps. Il existe plusieurs manières efficaces listées ci-dessous.

  • Si vous êtes proche d'une centrale nucléaire, quoi de mieux que de lancer le corps directement dans le réacteur.
    Qui vas réellement se sacrifier pour récupérer le corps?

  • Vous n'avez pas accès à une centrale nucléaire?
    Il y a toujours l'option de jeter le corps dans un chaudron industriel d'acide sulphurique.

  • Aucune de ces options est viable? Vous pouvez toujours vous débarasser du corps en le noyant dans le fond de l'océan.
    Surtout n'oubliez pas la brique!
