Corps

Se débarrasser du corps

L'un des plus gros problèmes rencontrés est de ce débarasser du corp. Il existe plusieur manière efficasse listé si dessous.

  • Si vous etes proche d'une centrale nucleaire, quoi de mieu que de lancer le corp directement dans le reacteur.
    Qui vas reelement ce sacrifier pour recupere le corp?

  • Vous n'avez pas acces a une centrale nuclaire?
    Il y a toujours l'option de jeter le corp dans un chaudron industriel d'acid sulphurique.

  • Aucune de ces options est viable? Vous pouvez toujours vous débarassez du corp en le noyant dans le font de l'ocean.
    Surtous n'oublier pas la brique!
